
# Bias-Free Text Checker

## ✅ Overview
The Bias-Free Text Checker is a simple system that detects biased or non-inclusive words in a given text and suggests neutral alternatives.  
The goal is to improve clarity, fairness, and inclusiveness in everyday communication.

---

## ✅ Problem Statement
Biased words such as “chairman,” “old man,” “guys,” or gender-specific pronouns are commonly used without realizing they may sound insensitive or exclusive.  
There is a need for a tool that automatically identifies such terms and recommends more neutral and inclusive language.

---

## ✅ Objectives
- Detect biased or insensitive words  
- Suggest neutral replacements  
- Highlight biased phrases clearly  
- Provide corrected text  
- Support voice input and voice output  
- Display real-time statistics and trends  
- Store analysis records for reference  

---

## ✅ Features
- Text bias detection  
- Neutral word suggestions  
- Highlighting of biased phrases  
- Original vs corrected text comparison  
- Voice input (speech to text)  
- Voice output (summary reading)  
- Analytics dashboard  
- Local record storage

## ✅ How It Works
1. User enters or speaks text  
2. System identifies biased words  
3. Neutral alternatives are suggested  
4. Original and corrected text are shown side-by-side  
5. Results are stored in a log  
6. Dashboard updates statistics
